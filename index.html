Native Android App - Cannot Run in Browser

This is a native Android application built with Kotlin and Jetpack Compose. It requires Android Studio and an Android device or emulator to run. The source code is available in this repository for viewing and editing.

To Run This App Locally:

  1. Clone this repository to your local machine
  2. Open in Android Studio (Arctic Fox or later)
  3. Sync Gradle files
  4. Run on an emulator or Android device (Android 10+)
